# Basic Python Honeypot

This project is a simple honeypot that listens for incoming connections on a specified port, logs the IP and data of any connection, and simulates a basic interaction.

## Features
- Logs all incoming connections
- Simulates an open, vulnerable port
- Helps analyze unauthorized access attempts
